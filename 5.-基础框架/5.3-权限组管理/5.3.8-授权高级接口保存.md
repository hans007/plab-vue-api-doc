# 5.3.8-授权高级接口保存

## URL

/base/permissionGroups/:id/apis

## 支持格式

JSON

## HTTP请求方式

PUT

## 是否需要登录

是

如果request中的认证信息不正确，要返回状态码 401，即NOT AUTHORIZED

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
id                    |   是   | number    | id
apis                  |   是   | array     | 授权 apis ids [1101,1102,1103]

## 注意事项

### HTTP状态

- 201 Created - 对创建新资源的 POST 操作进行响应。应该带着指向新资源地址的 Location 头

## 返回结果

返回实体数据

## 返回字段说明

参考请求主体参数
