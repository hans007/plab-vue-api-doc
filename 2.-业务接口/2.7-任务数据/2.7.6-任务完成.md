# 2.7.6-任务完成

## URL

/tasks/:id/complete

## 支持格式

JSON

## HTTP请求方式

PUT

## 是否需要登录

是

如果request中的认证信息不正确，要返回状态码 401，即NOT AUTHORIZED

## 请求参数

- 主体

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
uid                 |   是   | long    | 用户uid
id                  |   是   | long    | 数据id
results             |   是   | array   | 成果 {...}
attachments         |   是   | array   | 附件 {...}

- `results` 成果

字段 | 类型及范围 | 说明
----|----------|-------------
id                 | long       | id
name               | string     | 成果类型名称
estimate_count     | int        | 预估成果数量
actual_count       | int        | 实际成果数量
calculate_count    | int        | 系数 小计成功数 = weight * estimate_count
unit               | string     | 计量单位
weight             | int        | 成果系数

- `attachments` 附件

字段 | 类型及范围 | 说明
----|----------|-------------
fid             | string  | 文件唯一标识

## 注意事项

### HTTP状态

- 200 OK - 对成功的 GET、PUT、PATCH 或 DELETE 操作进行响应

## 返回结果

返回实体数据

## 返回字段说明

参考请求主体参数
