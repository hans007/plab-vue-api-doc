# 2.4.4-任务类型

## URL

/base/task_type

## 支持格式

JSON

## HTTP请求方式

GET

## 是否需要登录

是

如果request中的认证信息不正确，要返回状态码 401，即NOT AUTHORIZED

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------

## 注意事项

### HTTP状态

- 当GET 不到数据时，状态码要返回404，即NOT FOUND
- 200 OK - [GET]：服务器成功返回用户请求的数据

## 返回结果

```json
[
    {
        "id":101,
        "name":"研发"
    },
    {
        "id":102,
        "name":"维护"
    }
]
```

## 返回字段说明

字段 | 类型及范围 | 说明
----|----------|-------------
id              | number       | id
name            | string     | 任务类型名称
