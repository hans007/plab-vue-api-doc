# 2.2.3-更新面板排序

## URL

/dashboard/orders

## 支持格式

JSON

## HTTP请求方式

PUT

## 是否需要登录

是

如果request中的认证信息不正确，要返回状态码 401，即NOT AUTHORIZED

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
uid     | 是   | string  | 用户uid
data    | 是   | json    | 面板排序json { 类型:排序号, 类型:排序号 }

- data 参数json格式

```json
{
  'panel-lastTasks': 1,
  'panel-news': 2
}
```

## 注意事项

### HTTP状态

- 201 CREATED - [POST/PUT/PATCH]：新建或修改数据成功。

## 返回结果

```json
{
  'panel-lastTasks': 1,
  'panel-news': 2
}
```
