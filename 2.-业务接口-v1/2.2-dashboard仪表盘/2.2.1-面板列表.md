# 2.2.1-面板列表

## URL

/dashboard/list

## 支持格式

JSON

## HTTP请求方式

GET

## 是否需要登录

是

如果request中的认证信息不正确，要返回状态码 401，即NOT AUTHORIZED

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
uid | 是   | string  | 用户uid

## 注意事项

### HTTP状态

- 当GET 不到数据时，状态码要返回404，即NOT FOUND
- 200 OK - [GET]：服务器成功返回用户请求的数据

## 返回结果

```json
{
  "list": [{
    "type": "panel-news",
    "height": 300,
    "grid": 4
  }, {
    "type": "panel-projectStatus",
    "height": 300,
    "grid": 8
  }, {
    "type": "panel-lastTasks",
    "height": 300,
    "grid": 4
  }, {
    "type": "panel-myTasks",
    "height": 300,
    "grid": 8
  }, {
    "type": "panel-underwayTasks",
    "height": 300,
    "grid": 8
  }]
}
```

## 返回字段说明

字段 | 类型及范围 | 说明
----|----------|-------------
type   | string  | 面板类型
height | int     | 高度
grid   | int     | 1~12 宽度
