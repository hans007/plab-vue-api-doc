# 2.2.2-更新面板尺寸

## URL

/dashboard/size

## 支持格式

JSON

## HTTP请求方式

PUT

## 是否需要登录

是

如果request中的认证信息不正确，要返回状态码 401，即NOT AUTHORIZED

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
uid     | 是   | string  | 用户uid
type    | 是   | string  | 面板类型
grid    | 是   | int     | 宽度
height  | 是   | int     | 高度

## 注意事项

### HTTP状态

- 200 OK - 对成功的 GET、PUT、PATCH 或 DELETE 操作进行响应。

## 返回结果

```json
{
  "type": "panel-news",
  "height": 300,
  "grid": 4
}
```

## 返回字段说明

字段 | 类型及范围 | 说明
----|----------|-------------
type   | string  | 面板类型
height | int     | 高度
grid   | int     | 1~12 宽度
