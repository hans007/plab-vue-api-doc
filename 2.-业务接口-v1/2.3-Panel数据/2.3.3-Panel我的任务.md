# 2.3.3-Panel我的任务

## URL

/panels/mytasks

## 支持格式

JSON

## HTTP请求方式

GET

## 是否需要登录

是

如果request中的认证信息不正确，要返回状态码 401，即NOT AUTHORIZED

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
uid | 是   | string  | 用户uid

## 注意事项

### HTTP状态

- 当GET 不到数据时，状态码要返回404，即NOT FOUND
