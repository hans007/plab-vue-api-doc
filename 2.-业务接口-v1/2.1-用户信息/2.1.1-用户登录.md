# 2.1.1-用户登录

## URL

/users/login

## 支持格式

JSON

## HTTP请求方式

GET

## 是否需要登录

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
uname       | 是   | string  | 账号
upasswd     | 是   | string  | 密码

## 注意事项

## 返回结果

### 成功

```json
{
    "isok": true,
    "user": {
        "uid": "91002",
        "name": "吴雪",
        "screen_name": "北总研究院-研究员-吴雪"
    }
}
```

### 错误

```json
{
    "isok": false,
    "err": {
        "code" : 20003,
        "message" : "用户登录失败",
        "description" : "用户不存在"
    }
}
```

## 返回字段说明

字段 | 类型及范围 | 说明
----|----------|-------------
isok            | string  | 是否登录成功
uid             | string  | 用户uid
name            | string  | 用户姓名
screen_name     | string  | 友好显示，带部门职位

## 错误码

错误代码 | 错误信息
--------|---------
10013   | 被系统屏蔽的用户
20003   | 用户不存在
21302   | 用户名或密码不正确
