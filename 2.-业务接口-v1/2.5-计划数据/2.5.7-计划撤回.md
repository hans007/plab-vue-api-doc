# 2.5.7-计划撤回


## URL

/plans/:id/recall

## 支持格式

JSON

## HTTP请求方式

PUT

## 是否需要登录

是

如果request中的认证信息不正确，要返回状态码 401，即NOT AUTHORIZED

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
uid                 |   是   | long    | 用户uid
id                  |   是   | long    | 数据id

## 注意事项

### HTTP状态

- 200 OK - 对成功的 GET、PUT、PATCH 或 DELETE 操作进行响应

## 返回结果

返回详情数据

## 返回字段说明

见详情设计
