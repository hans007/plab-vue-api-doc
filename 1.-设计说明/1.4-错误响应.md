# 1.4-错误响应

## 响应体

## 标准返回

- 唯一的错误码
- 有用的错误信息
- 可能的话，提供错误细节的描述

```json
{
  "code" : 1234,
  "message" : "数据保存错误",
  "description" : "违反表唯一规则 pcode "
}
```

## 多条信息

```json
{
  "code" : 1024,
  "message" : "Validation Failed",
  "errors" : [
    {
      "code" : 5432,
      "field" : "first_name",
      "message" : "First name cannot have fancy characters"
    },
    {
       "code" : 5622,
       "field" : "password",
       "message" : "Password cannot be blank"
    }
  ]
}
```
